# 初始化项目

## 初始化
npm
```sh
npm create vue@latest 

# 等价于 npx create-vue@latest
# 或者 npm init vue@latest
```


```sh
pnpm create vue@latest
```

```sh
# For Yarn (v1+)
yarn create vue
```
```sh
# For Yarn Modern (v2+) ,更好的缓存机制
yarn create vue@latest
```
  
```sh
# For Yarn ^v4.11
yarn dlx create-vue@latest
```
